## 使用方法

在`docsDirectory`的主目录下创建一个名为`index.json`的文件，即可自定义排序。文件内容如下：

```json
[
  "README.md",
  {
    "title": "使用教程",
    "docs": [
      "使用教程/目录菜单排序须知.md",
      "使用教程/Markdown编写须知.md"
    ]
  },
  {
    "title": "语法支持示例",
    "docs": [
      "文档相关/Markdown语法样例.md",
      "文档相关/Mermaid图表样例.md"
    ]
  },
  "README_EN.md"
]
```

## 注意

1. 文件夹的`title`、`docs`字段为必填项，且`docs`字段中的相对位置必须与实际文件夹路径和文件名一致。

2. 本排序功能仅运行使用两层文件结构，默认排序支持多层文件结构。

3. 如果`index.json`文件不存在，默认按照文件夹名称排序。

4. 如果`index.json`文件存在，格式不符合要求，将按照默认排序规则排序。

5. 构建只对`index.json`里有的文件进行排序，没有写上就不会构建。
