## 使用方法

在`docsDirectory`的主目录下创建一个名为`index.json`的文件，即可自定义排序。文件内容如下：

```json
[
  "README.md",
  {
    "title": "使用教程",
    "path": "使用教程",
    "docs": [
      "Markdown编写须知.md"
    ]
  },
  {
    "title": "语法支持示例",
    "path": "文档相关",
    "docs": [
      "Markdown语法样例.md",
      "Mermaid图表样例.md"
    ]
  },
  "README_EN.md"
]
```

## 注意

1. 文件夹的`title`、`path`、`docs`字段为必填项，且`path`字段和`docs`字段中的文档文件名必须与实际文件夹路径和文件名一致。

2. 本排序功能仅限于对当前目录下的文件/文件夹进行自定义排序，不支持将文件移动到其他目录层级然后再排序。

3. 如果`index.json`文件不存在，默认按照文件夹名称排序。

4. 如果`index.json`文件存在，但其中的文件夹路径或文档文件名与实际不符，将按照默认排序规则排序。
